Banco de Dados - 27/02/23


Qual a diferença entre dados e informações?
	- Dados são fatos crús. Informações são o processamento de dados crús para revelar algum significado

O que é um Banco de Dados?
	- Conjunto de dados integrados que tem por objetivo atender a uma comunidade de usuários

Qual a função de um Banco de Dados?
	- Organizar a estrutura de informações, facilitando consultas e atualizações

A maioria dos BDs estão atrlados a um SGBD. O que é um SGBD?
	- Um conjunto de softwares que permitem armazenar. modificar e extrair dados de um banco.

Pelo que os SGBDs são responsáveis?
	- Intermediar os processos de construção, manipulação e administração do banco de dados solicitados por usuários ou aplicações


Banco de Dados Centralizados:
	- Dados alocados em um único local

Banco de Dados Distribuidos:
	- Dados distribuidos em diferentes locais.

Banco de Dados Operacionais: 
	- Suporte as operações do dia a dia

Transacional:
	- Banco de dados de produção

Data Warehouse:
	- Armazena dados usados para decisões estratégicas e táticas

Dados não-estruturados:
	- Existem na forma original, sem formatação

Dados Estruturados:
	- Resultante da formatação dos dados
